# Macrotrends.net 竞品分析 & X-Gudao 超越方案

## 一、Macrotrends 核心模式分析

### 1.1 网站定位
- **口号**: "The Long Term Perspective on Markets"
- **核心**: 长期历史数据可视化（50年+）
- **数据类型**: 股票、指数、大宗商品、利率、汇率、经济指标

### 1.2 核心功能模块

| 模块 | 功能 | 技术实现 |
|------|------|----------|
| **Stock Screener** | 股票筛选器 | 多维度筛选条件 |
| **Interactive Charts** | 交互式图表 | Highcharts/D3.js iframe嵌入 |
| **Historical Data** | 历史数据下载 | CSV导出 |
| **Market Indices** | 市场指数 | 标普500、道琼斯等长期走势 |
| **Global Metrics** | 全球经济指标 | GDP、通胀、利率等宏观数据 |

### 1.3 数据展示特点
- **时间跨度**: 极长（部分数据从1960s开始）
- **图表类型**: 折线图为主，支持缩放
- **数据导出**: 支持CSV下载
- **嵌入方式**: iframe嵌入 (`chart_iframe_comp.php?id=xxx`)

### 1.4 商业模式
- 免费基础数据
- 广告收入
- 可能的数据订阅服务

## 二、Macrotrends 的局限性（X-Gudao 的机会）

| 局限 | 说明 | X-Gudao 差异化 |
|------|------|----------------|
| ❌ **无AI分析** | 纯数据展示，无智能解读 | ✅ **AI深度洞察** - 本地Ollama生成分析 |
| ❌ **英文为主** | 无中文界面 | ✅ **中文原生** - 全中文体验 |
| ❌ **无ETF对比** | 单资产展示，无对比功能 | ✅ **多资产对比** - VOO/QQQ/SPY一键对比 |
| ❌ **无风控提示** | 无投资风险提示 | ✅ **风控内置** - 强制免责声明 |
| ❌ **数据延迟** | 非实时数据 | ✅ **Yahoo Finance实时** - 近实时数据 |
| ❌ **无技术分析** | 无MA/RSI/MACD等指标 | ✅ **自动技术指标** - 一键计算 |
| ❌ **隐私问题** | 数据上传云端 | ✅ **本地部署** - 隐私优先 |
| ❌ **交互弱** | 静态图表 | ✅ **Plotly交互** - 可缩放、悬停查看 |

### 3.4 视觉设计方向

**Macrotrends 风格**: 传统金融网站，蓝白配色，信息密集

**X-Gudao 风格**: 
- 🎨 **现代简约**: 留白更多，呼吸感
- 🌙 **深色模式**: 默认深色（护眼，适合长时间看图表）
- 📊 **卡片式布局**: 信息模块化
- ✨ **微交互**: 悬停效果、加载动画
- 🎯 **AI标识**: 突出AI分析的独特价值

### 3.5 技术架构优势

| 层级 | Macrotrends | X-Gudao |
|------|-------------|---------|
| 前端 | 传统jQuery + Highcharts | 现代HTML5 + Plotly.js |
| 后端 | PHP (推测) | Python + FastAPI (可选) |
| AI | ❌ | Ollama本地大模型 |
| 数据 | 自建数据库 | Yahoo Finance API |
| 部署 | 传统服务器 | GitHub Pages + Actions |
