📈 关于股道奇货

AI驱动的金融数据分析平台，让投资决策更聪明

🎯 项目介绍

股道奇货 X-Gudao 是一个开源的AI驱动金融数据分析平台。 我们致力于为投资者提供专业的量化分析工具，包括实时行情数据获取、交互式K线图表、 技术指标计算、多资产对比分析等功能。 平台融合了大语言模型（LLM）的智能洞察能力，为用户提供更深入的维度分析。

✨ 核心特性

📡

实时数据

通过 yfinance 获取全球股票、ETF、指数的实时和历史行情数据，支持多种时间周期。

📊

交互式图表

基于 ECharts 的专业K线图，支持缩放、平移、十字光标、均线叠加等交互功能。

🤖

AI 洞察

集成 Ollama 本地大语言模型，提供智能市场分析和投资建议（敬请期待）。

🇨🇳

中文原生

专为中文用户设计，界面、提示、分析报告均为中文，降低使用门槛。

🛠️ 技术栈

FastAPI Python yfinance ECharts Ollama HTML/CSS/JS
